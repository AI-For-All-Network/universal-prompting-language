# Contributing to Universal Prompting Language

Thank you for your interest in contributing.

## How to contribute

- Fork the repository and open a pull request against main.
- Follow the coding and documentation style used in the repo.
- Include tests for new features or bug fixes.

## Affiliation note

If you are contributing on behalf of an organization, please state your affiliation in your pull request or contributor profile. If you represent AI For All Network in the future, notify maintainers so repository ownership and legal notices can be updated.
