# Universal Prompting Language

Universal Prompting Language UPL is a model-agnostic, human-centered prompting standard designed to make prompts predictable, reusable, and safe for everyday users and developers.

## Quick summary

UPL provides a simple syntax, a clarification-first interaction model, and a schema-driven approach so everyday users can get consistent results across different AI systems while developers can build reference implementations and integrations.
